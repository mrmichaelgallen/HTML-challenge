HTML Drill

What is the challange?

This page has been created using some of the most common HTML and CSS errors I have seen, and some of the worst practices.

This page uses out-dated code practices that are no longer considered best practice, and it definitely doesn't look like the mock-up provided.

Your Job

You need to get this page up and running.

  1. Remove tags that have been removed from the HTML5 standard. See this handy list from W3Schools for what tags have been removed from HTML5. Using the new semantic tags like <article>, <section>, <aside>, <header>, and <footer> would be ideal, but you may also use <div> tags if you prefer.
  2. Update to best styling practices. Remove all inline styles used and create / link an external style sheet for all necessary CSS.
  3. Make it responsive. Most sites are viewed on mobile devices now, so we need to create websites that have been optimized for viewing on mobile as well as desktop. I would suggest using @media queries, but you can do it anyway you like.
  4. Does it validate?Make sure your page will validate using W3C Markup Validation Service

Want to do more?

Getting the page layout to match the design mock-up and getting the code to validate is your main goals on this Code Challange. If you finish it and want to expand on this project, here are some ideas:

  1. Use a framework like Foundation or Bootstrap to build the site out.
  2. Create a matching "About" page.
  3. Create a matching "Contact" page with a form that users can use to send a quick message, as well as links to connect on social media like LinkedIn or Facebook.
  4. Create a matching "Gallery" page to display pictures.
  5. If you have already gotten to the JavaScrpt and jQuery courses, you could integrate your image slider into the hero image on this homepage, or turn the side navigation into a drop-down menu.
